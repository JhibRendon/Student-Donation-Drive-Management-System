# Student Donation Drive Management System

## Overview
The **Student Donation Drive Management System** is a web-based platform designed to simplify and organize the process of creating, approving, monitoring, and documenting donation campaigns. It allows administrators to efficiently manage drives while providing donors an accessible way to participate and track their contributions. Users can submit donation drives by entering necessary campaign information for proper documentation and faster review.  

Key system functionalities include:
- Managing donation campaigns  
- Tracking donor contributions  
- Generating reports  
- Sending notifications  
- Maintaining activity logs for transparency  

The system uses **Role-Based Access Control (RBAC)** to ensure only authorized users can access specific parts of the system and **Multi-Version Concurrency Control (MVCC)** to prevent database conflicts when multiple users access or update records simultaneously.

## Technology Stack
- **Frontend:** Typscript & React JS (MERN STACK)  
- **Backend:** Node.js   
- **Database:** MongoDB Atlas  
- **Tools & Frameworks:** Trello, APIs, external services  
- **Security:** Authentication, RBAC, Data protection  

## Features
- Create, approve, and manage donation campaigns  
- Track donor contributions and participation  
- Generate detailed reports for campaigns and donations  
- Send notifications and reminders to users  
- Maintain activity logs for transparency  
- Role-based access control for data security  
- MVCC support for concurrent database access
